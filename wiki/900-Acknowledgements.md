# Acknowledgements

## K3NG CW Keyer v1

Version 2 is a ground-up rewrite, but it would not exist without the decade-plus of development that went into v1. The v1 feature set, user community, and field experience directly shaped the design goals and feature selection of v2.

## Open Source Community

This project is built on the shoulders of the Arduino community and the many contributors to open source amateur radio software. The Arduino platform makes projects like this accessible to everyone from beginners to experienced embedded developers.
